Plain English explanation:

Filtration - available information about the process, can be considered as logs. Natural filtration - contains all the information about the given process, and only this information.
Process is adapted to filtration if the information about the state of the given process at time \(t\) contains at \(\sigma\)-algebra \(\mathcal{F}\).

Q: Why do we need a \(\sigma\)-algebra when talking about stochastic processes? A: Just to restrict ourselves to the case when all the events we are talking about can be assigned a probability measure
